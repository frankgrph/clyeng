香港轉車搜尋 🚌

輸入起點同終點,自動計算實時轉車組合 · 参考九巴/城巴/嶼巴/專線小巴/港鐵巴士 API

九巴 KMB 城巴 CTB 嶼巴 NLB 專線小巴 GMB 港鐵巴士 MTRB
關於此工具：本工具使用 data.gov.hk 提供嘅九巴/龍運(data.etabus.gov.hk)、城巴(rt.data.gov.hk)、嶼巴、專線小巴同港鐵巴士實時到站 API，並以 hkbus/hk-bus-crawling 提供嘅每日路線靜態資料庫作為路網基礎。聯運路線(例如九巴同城巴共同營運)會同時檢索多營辦商 API 數據以避免「盲區」遺漏。車程所需時間將優先對比運輸署 `stop_times.txt` 排班表各站點間的預定行車時間進行估算，若對應班次同時有上車及落車實時到站數據，會以實時差值校準。若沒有落車實時 ETA，會沿途比對上車班次的相鄰站實時到站 ETA 進行單邊校準，並可透過前一班次的實際車程進一步校正；若該路線未有靜態班次時間表覆蓋，則會降級使用地圖拓樸物理距離與預設速度估計車程並進行相同邏輯的校準，僅供參考，實際車程可能因交通燈、路權、上落客人數等有出入。轉乘步行距離用直線距離估算(≤300米)，起訖點會自動搜尋≤250米内其他巴士站作替代。搜尋只會保留現正有實時班次嘅路線組合，轉車腿必須喺一小時内有實時到站資料先會成立；若某腿冇實時到站數據要靠估計班次，會再用運輸署「公共交通服務班次資料」(headway)核實嗰個時段係咪真係有服務，過濾未有服務嘅繁忙時間專線。
