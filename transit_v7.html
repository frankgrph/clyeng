香港轉車搜尋 🚌

輸入起點同終點,自動計算實時轉車組合 · 参考九巴/城巴/嶼巴/專線小巴 API

九巴 KMB 城巴 CTB 嶼巴 NLB 專線小巴 GMB
關於此工具：本工具使用 data.gov.hk 提供嘅九巴/龍運(data.etabus.gov.hk)、城巴(rt.data.gov.hk)、嶼巴同專線小巴實時到站 API,並以 hkbus/hk-bus-crawling 提供嘅每日路線靜態資料庫作為路網基礎。聯運路線(例如九巴同城巴共同營運)會同時檢索多營辦商 API 數據以避免「盲區」遺漏。冇實時到站數據可用嘅車程,優先以運輸署 TDAS 路況預測 API(基於智能道路網絡 IRN 同實際路況數據)計算真實路徑距離同行車時間,並加上巴士埋站上落客嘅停留時間;若 TDAS 查唔到(例如超出90分鐘預測範圍或連線失敗),先會回退用直線距離推算,僅供參考,實際車程可能因交通燈、路權、上落客人數等有出入。轉乘步行距離用直線距離估算(≤300米),起訖點會自動搜尋≤250米内其他巴士站作替代。搜尋只會保留現正有實時班次嘅路線組合,轉車腿必須喺一小時内有實時到站資料先會成立;若某腿冇實時到站數據要靠估計班次,會再用運輸署「公共交通服務班次資料」(headway)核實嗰個時段係咪真係有服務,過濾未有服務嘅繁忙時間專線。
